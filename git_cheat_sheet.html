Git Cheat Sheet

Here we have an overview of the most popular functions in Git.
We'll cover the following frequent tasks

  • Installing Git
  • Creating a repository
  • Adding files to the staging area
  • Commit to a repository
  • Commit to a remote repository
  • Status and log
  • Go back in time
  • Branching
  • Merging

Creating a Repository

If you want to create a repository of a project, place all the files if the project in the same folder. In you console, make this file your working directory. Once you are in that directory type:
user$ git init You'll see a message saying that a repository was created in your folder

Adding files to the staging area

If you want to Git to keep track of a file, you need to add it to the staging area. To accomplish that, type:
user$ git add filename Also you can add all the files in your folder typing
user$ git add .

Commit to a repository

A commit is like a snapshop of the files you have in the staging area in a particular moment
user$ git commit Also you can add all the files in your folder typing
user$ git add .
